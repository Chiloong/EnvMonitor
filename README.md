# EnvAlert

Monitor environmental changes in **pressure and wind** with real-time notifications.

- Tracks **pressure change rate (ΔP/Δt)** for trend alerts  
- Monitors **wind speed and direction** with threshold alerts  
- Sends notifications via **Bark**  
- Supports historical data storage and trend calculation  
- Designed for future integration with other environmental data sources (IoT, sensors, etc.)
